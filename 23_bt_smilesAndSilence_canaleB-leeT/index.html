Features and Components


This Site

This site contains a number of features from Bootstrap that are actively in use. These include:

  • A Responsive Grid - When you change the width of your browswer screen, the menu will pop up as a top bar as opposed to a vertical navigation bar.
  • Navigation Bar - The navigation bar contains functional links to other parts of this site and external sources.
  • A Sticky Footer - Using a combination of inline CSS and Bootstrap grids, a footer can be made sticky so that it is always visible.
  • An Alert - The alert above provides information about the website, as well as containing an alert-link. It is not dismissible.

    Jumbotrons

    Jumbotrons can be used to showcase key messages on a site since they occupy the most space they can.

    JUMBOTRON!!!

    This is a jumbotron

    It's pretty jumbo

    Learn more 
    
              <div class="jumbotron">
		<h1 class="display-4">JUMBOTRON!!!</h1>
		<p class="lead">This is a jumbotron</p>
		<hr class="my-4">
		<p>It's pretty jumbo</p>
		<a class="btn btn-primary btn-lg" href="#" role="button">Learn more</a>
	      </div>

    FLUID JUMBOTRON!!!

    A fluid jumbotron occupies the entire horizontal space of its parent and also lacks rounded corners.


    
              <div class="jumbotron jumbotron-fluid">
		<div class="container">
		  <h1 class="display-4">FLUID JUMBOTRON!!!</h1>
		  <p class="lead">A fluid jumbotron occupies the entire horizontal space of its parent and also lacks rounded corners.</p>
		</div>
	      </div>


    Pagination

    Pagination can be used to navigate a site with multiple pages.

    • Previous
    • 1
    • 2
    • 3
    • Next

    
              <nav aria-label="Pagination">
		<ul class="pagination">
		  <li class="page-item"><a class="page-link" href="#">Previous</a></li>
		  <li class="page-item"><a class="page-link" href="#">1</a></li>
		  <li class="page-item"><a class="page-link" href="#">2</a></li>
		  <li class="page-item"><a class="page-link" href="#">3</a></li>
		  <li class="page-item"><a class="page-link" href="#">Next</a></li>
		</ul>
	      </nav>

    You can make a certain page active, showing what page the user is on.

    • Previous
    • 1
    • 2
    • 3
    • Next

    
              <li class="page-item active"><a class="page-link" href="#">1</a></li>


    You can also make a page disabled, meaning the user cannot go there from where they are currently.

    • Previous
    • 1
    • 2
    • 3
    • Next

    
              <li class="page-item disabled"><a class="page-link" href="#">Previous</a></li>
